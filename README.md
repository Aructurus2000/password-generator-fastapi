# Password Generator API (FastAPI)

Простой и быстрый сервис для генерации безопасных и уникальных паролей, написанный на **Python** с использованием **FastAPI**.

## Основные возможности
- Генерация любого количества паролей за один запрос.
- Настройка длины пароля (от 8 до 100 символов).
- Гарантированное наличие цифр, строчных и заглавных букв (исключая неоднозначные символы `O, I, l, 0`).
- Автоматическая документация API (Swagger UI).

## Установка и запуск

1. **Клонируйте репозиторий:**
   git clone https://github.com
   cd password-generator-fastapi
   
   **Установите зависимости:**
   pip install -r requirements.txt
   
   **Запустите сервер:**
   uvicorn generate_pass:app --reload
   
   **Генерация 5 паролей длиной 10 символов:**
   http://127.0.0.1:8000/generate?count=5&length=10

   ### Интерактивная документация (Swagger UI)
Самый удобный способ протестировать API без ручного ввода адресов использовать встроенную документацию:
  **[http://127.0.0.1:8000/docs](http://127.0.0.1:8000/docs)**
